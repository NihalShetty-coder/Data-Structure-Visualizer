

# Data Structure Visualizer

An interactive web application for visualizing and understanding core data structures and their operations. Built with React, Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Features

- Visualize common data structures: Array, Linked List, Stack, Queue, Binary Tree, Graph
- Step-by-step animations for traversals and operations (push, pop, enqueue, dequeue, etc.)
- Mock code editor showing example code for each structure
- Control panel to adjust animation speed and control playback
- Responsive, modern UI with light/dark mode

## Getting Started

To run locally:

```sh
git clone https://github.com/NihalShetty-coder/Data-Structure-Visualizer.git
cd Data-Structure-Visualizer
npm install
npm run dev
```

Open http://localhost:8080 in your browser.
