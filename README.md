# Data Structure Visualizer

An interactive web application that helps users understand fundamental data structures through real-time visualization and step-by-step animations. Designed for students, educators, and interview preparation, the platform simplifies complex concepts by visually demonstrating how operations work internally.

The application focuses on clarity, performance, and an intuitive learning experience using a modern frontend architecture.

---

## 🚀 Core Features

- Visualizes core data structures: **Array, Linked List, Stack, Queue, Binary Tree, and Graph**
- Step-by-step animations for traversals and operations (**push, pop, enqueue, dequeue, insert, delete, search**)
- Interactive control panel to **adjust animation speed** and **control playback**
- Mock code editor displaying **example code snippets** for each data structure
- Responsive and modern UI with **light / dark mode support**
- Designed to improve **DSA conceptual clarity and interview readiness**

---

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **UI Components:** shadcn-ui (Radix UI primitives)
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Visualization:** D3.js / SVG / Canvas
- **Animation:** Framer Motion
